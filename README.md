# iAccessibility Website

The [iaccessibility.net](https://iaccessibility.net) website — *Making Success Accessible*.

This repository hosts the **Next.js** rebuild of the iAccessibility website, migrating
from the existing WordPress.com site. iAccessibility is a community-driven platform for
accessible technology, covering:

- **iAccessibility Report** — in-depth reviews of accessible apps, devices, and tech products
- **iACast Network** — podcasts on accessible technology
- **App directories** — crowdsourced accessibility info across platforms

Accessibility is not a feature here — it is the product. All UI must meet WCAG 2.2 AA.

This repository covers the **website** only. A dedicated iAccessibility app will follow
in a separate repository.
